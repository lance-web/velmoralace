# Velmora Lace optimized static website

Upload all files to your GitHub repository root and deploy with Cloudflare Pages.

Important:
1. Keep `index.html` in the repository root.
2. Keep the `assets` folder.
3. Submit `/sitemap.xml` to Google Search Console.
4. Test the Web3Forms inquiry form and catalog form.
5. Later, move external images to Cloudflare R2 or local `/assets` for better speed and stability.
